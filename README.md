# QuizApp (skeleton)

This is a minimal Android Studio Java project skeleton for a Quiz app with Text-To-Speech support.

Open the `QuizApp` folder in Android Studio and let it sync/upgrade Gradle if needed.

Files included:
- MainActivity (launcher)
- QuizActivity (question UI + TTS)
- Question.java and QuestionRepository.java (sample questions)
- layouts and manifest

You may need Android Studio to download gradle plugin and wrappers on first open.
